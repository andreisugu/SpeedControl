# Support Guide

Welcome to the `SpeedControl` support guide. If you have questions, encounter bugs, or need assistance configuring the plugin, please use the channels described below.

## Support Channels

### 1. GitHub Discussions
For general questions, configuration assistance, sharing ideas, and showcasing setups, please open a thread in **GitHub Discussions** under the Q&A or General category.

### 2. GitHub Issues
If you have verified a bug or want to suggest a new feature:
- Search existing issues to see if it has already been reported.
- If not, create a new issue using the provided templates, including details about your Pumpkin server version, configuration, and log outputs.

### 3. Documentation
Please check our documents first:
* [README.md](README.md) - Installation, usage commands, permissions, and features.
* [ARCHITECTURE.md](ARCHITECTURE.md) - Code layout and SOLID modules design.
* [WORKFLOW.md](WORKFLOW.md) - Local compilation, deployment, and permission pre-approvals troubleshooting.
